Type Alias: ITrackingLayer

ITrackingLayer = object

跟踪图层

可以存放临时数据，例如：存放搜索结果，绘制对象（实线，虚线，曲线，矩形，圆，多边形），编辑对象（新增节点，删除节点，移动节点）
